Vantagens do Git Hub

Hoje em dia GitHub é essencial, vejo cada vez mais as empresas pedindo pelo menos o conhecimento em Git/GitHub, que são ótimas ferramentas de controle de versão.

O GitHub estende a ferramenta Git com uma ampla gama de recursos colaborativos e oferece hospedagem gratuita para projetos open source. Os desenvolvedores do kernel do Linux utilizaram o serviço como sua principal plataforma de colaboração durante várias semanas em 2011, no período em que o site kernel.org esteve fora do ar devido a um ataque hacker. O GitHub fornece a habilidade de criar repositórios privados para software proprietário e também oferece o serviço GitHub Enterprise, que permite que clientes instalem sua própria versão local do software.
